# Enunciado del proyecto

## Objetivos
- Diseñar un plan de juego básico para principiantes en pádel.
- Identificar y explicar los elementos clave del campo de juego.

## Contenido
Para este módulo final, se centrará en la elaboración de un proyecto que permita a los estudiantes aplicar todo lo aprendido durante el curso. El objetivo es crear un plan estratégico básico para principiantes en pádel, incluyendo una descripción detallada de los elementos clave del campo de juego y cómo utilizarlos efectivamente.

El plan debe abordar aspectos como la posición inicial de los jugadores, las zonas del campo (doble zona, media cancha, doble media cancha), y estrategias básicas para cada situación. Los estudiantes deben considerar cómo estos elementos influyen en la toma de decisiones durante el juego.

## Ejercicio
Cada estudiante deberá diseñar un plan de juego básico que incluya:
- Una descripción de su posición inicial en el campo.
- Las zonas del campo donde se moverá y por qué.
- Estrategias para cada zona: defensa, ataque y transición entre ambas.

El plan debe ser presentado en forma de diagrama o mapa táctico junto con una breve explicación escrita.

## Resumen
- Diseño de un plan estratégico básico para principiantes en pádel.
- Identificación y descripción de los elementos clave del campo de juego.
